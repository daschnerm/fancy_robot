fancy_robot implementation for [ROS](https://www.ros.org/)


## Setup

1. Pull repo into \src directory
2. catkin_make
3. source $workspace\devel\setup.bash
4. rosrun fancy_robot robocontrol.py

## Record setup

- Set self.desired_speed in init() to 0.85
